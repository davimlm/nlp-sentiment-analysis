# 🎵 NLP and Sentiment Analysis Project with Python

This project was developed to analyze emotional and linguistic patterns in Brazilian song lyrics released between 2015 and 2019 using Natural Language Processing (NLP), Machine Learning, and data visualization techniques.

The system processes song lyrics, structures datasets into JSON format, applies sentiment analysis using Transformer-based models, and generates visual insights such as word frequency analysis and word clouds.

---

# 🚀 Features

* Automated song lyric structuring into JSON format
* Text preprocessing and cleaning
* Word frequency analysis
* Sentiment and emotion classification
* Translation support for lyrics
* Data visualization with histograms and word clouds
* NLP pipeline using Transformer-based Deep Learning models

---

# 🧠 Technologies Used

* Python
* JSON
* Hugging Face Transformers
* RoBERTa
* NumPy
* Matplotlib
* WordCloud

---

# 📊 Project Overview

The project analyzes a collection of Brazilian songs from different genres and years between 2015 and 2019. Through NLP techniques and pre-trained Machine Learning models, the system identifies recurring emotional patterns, word usage trends, and sentiment distributions across the dataset.

A custom Python script was also developed to automate the conversion and organization of song lyrics into JSON format, enabling scalable text processing and analysis.

---

# 🤖 Machine Learning & NLP

The sentiment analysis pipeline was implemented using pre-trained Transformer-based models from Hugging Face, specifically the RoBERTa architecture for emotion classification.

Main concepts explored in the project:

* Natural Language Processing (NLP)
* Sentiment Analysis
* Deep Learning
* Transformers
* Text Processing
* Data Analysis
